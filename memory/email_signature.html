📝 Comment installer cette signature dans Apple Mail

  1. Sélectionne tout le bloc gris ci-dessous (Cmd + A sur la zone).
  2. Copie (Cmd + C).
  3. Ouvre Mail → Réglages → Signatures.
  4. Sélectionne ton compte, clique le +, donne-lui un nom.
  5. Décoche « Toujours utiliser la police par défaut ».
  6. Colle (Cmd + V) dans la zone de signature. Voilà. ✦
